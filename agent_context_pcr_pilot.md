## What the Project Is

**PCR Pilot** is a mobile-first demo ePCR (Electronic Patient Care Report) app for ambulance/EMT crews. It allows EMTs to create patient care reports using camera OCR and voice transcription, and supervisors to review them.

**Tech Stack:**
- Next.js 14 (App Router, TypeScript)
- Tailwind CSS
- Supabase (Postgres + Storage)
- OpenAI — Whisper (voice transcription) + GPT-4o-mini (OCR field extraction + narrative)
- Tesseract.js — primary OCR engine
- Puppeteer + @sparticuz/chromium — server-side PDF export
- OpenStreetMap Nominatim — address autocomplete

**Important:** This is a DEMO app — not HIPAA compliant, uses plaintext demo passwords, sample data only.
